## ⚡ The Vibe
Most task apps are cluttered. **Momentum** is designed to be the swiss knife of utility tools integrated seamlessly while keeping you ahead of your schedule. No bloat, No trackers, No ads. Just progress.

## 🎮 Features
* **Stat-Based Growth:** Every task completed contributes to your daily "XP."
* **Minimalist UI:** High-contrast dark mode designed for focus.
* **Habit Streaks:** Visual indicators to keep your momentum alive.
* **Lightweight:** Fast loading times, zero lag.

## 🛠️ Built With
* **HTML5 / CSS3:** Custom-built UI logic.
* **JavaScript:** Dynamic task handling and stat calculations.
* **GitHub Actions:** Automated APK building and distribution.

## 📥 Installation
1. Go to the **Releases** section of this repository.
2. Download the latest `Momentum vX.X.apk` file.
3. Install on your Android device (ensure "Install from Unknown Sources" is on).
